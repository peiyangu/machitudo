# 画像フォルダ構造

このフォルダには、まちつどWebサイトで使用する画像を配置します。

## フォルダ構成

```
images/
├── hero/          # トップページのメインビジュアル
├── about/         # イベント紹介・過去の写真
├── stores/        # 店舗の写真
│   ├── sweets-sample-1.jpg
│   ├── sweets-sample-2.jpg
│   ├── sweets-sample-3.jpg
│   ├── drink-sample-1.jpg
│   ├── drink-sample-2.jpg
│   ├── drink-sample-3.jpg
│   ├── food-sample-1.jpg
│   ├── food-sample-2.jpg
│   ├── food-sample-3.jpg
│   ├── handmade-sample-1.jpg
│   ├── handmade-sample-2.jpg
│   ├── handmade-sample-3.jpg
│   ├── workshop-sample-1.jpg
│   ├── workshop-sample-2.jpg
│   ├── workshop-sample-3.jpg
│   ├── festival-sample-1.jpg
│   ├── festival-sample-2.jpg
│   └── festival-sample-3.jpg
└── access/        # アクセス・マップ
```

## 画像の推奨サイズ

### 店舗カード画像
- **サイズ**: 640px × 480px (4:3)
- **フォーマット**: JPG
- **ファイルサイズ**: 100KB以下推奨
- **最適化**: TinyPNGなどで圧縮推奨

### メインビジュアル
- **サイズ**: 1920px × 1080px
- **フォーマット**: JPG
- **ファイルサイズ**: 300KB以下推奨

## 画像の準備方法

1. 各店舗の写真を準備（InstagramやWebサイトから）
2. 画像編集ツールで推奨サイズにリサイズ
3. TinyPNG (https://tinypng.com/) で圧縮
4. 適切なフォルダに配置

## 注意事項

- 画像の著作権に注意してください
- 店舗さんから許可を得た画像のみ使用してください
- ファイル名は半角英数字とハイフンのみ使用
- 日本語のファイル名は避けてください
